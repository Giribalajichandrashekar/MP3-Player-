# 🎵 MP3 Player


A clean, responsive, and feature-rich browser-based MP3 Player built using **HTML**, **CSS**, and **JavaScript**.

<p>
    <img src="https://img.shields.io/badge/HTML-5-orange?style=for-the-badge&logo=html5">
    <img src="https://img.shields.io/badge/CSS-3-blue?style=for-the-badge&logo=css3">
    <img src="https://img.shields.io/badge/JavaScript-ES6-yellow?style=for-the-badge&logo=javascript">
</p>

This project allows users to create playlists from their local audio files, control playback, shuffle tracks, and automatically stop playback using a built-in countdown timer—all without requiring any backend or external libraries.

---

# ✨ Features

- 🎧 Play local audio files directly from your computer
- 📂 Multiple file playlist support
- ⏯️ Play / Pause controls
- ⏮️ Previous Track
- ⏭️ Next Track
- 🔀 Shuffle Playlist
- ⏲️ Built-in Sleep Timer
- 🎵 Automatic next-song playback
- 🎯 Highlight currently playing track
- ⌨️ Keyboard shortcuts (Left / Right Arrow keys)
- 💻 Responsive user interface
- 🌈 Modern animated neon UI

# 🎮 How to Use

1. Open the application.
2. Click **Choose Files**.
3. Select one or multiple audio files.
4. Click **Play**.
5. Navigate songs using:

- Previous
- Next
- Shuffle

6. Set a custom timer if you want playback to stop automatically.

# 📌 Functionalities

### Playlist Management

- Load multiple songs
- Click any song to play instantly
- Highlight active track

### Playback Controls

- Play
- Pause
- Previous
- Next
- Automatic next track

### Shuffle Mode

Randomizes the playlist and starts playback from the shuffled list.

### Sleep Timer

Set Hours, Minutes, and Seconds.

When the timer reaches zero:

- Audio stops automatically
- Playback pauses
- Timer resets
